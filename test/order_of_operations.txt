add = sub < mul = div

- add = sub
	- (1 - 2) + 3 --> 1 - 2 + 3
	- 1 + (2 - 3) --> 1 + 2 - 3
	- (1 + 2) - 3 --> 1 + 2 - 3
	- 1 - (2 + 3) --> 1 - (2 + 3)
	- Summary: ((1 + 2) - 3) + (1 - (2 + 3))
	           --> 1 + 2 - 3 + 1 - (2 + 3)
- add < mul
	- (1 * 2) + 3 --> 1 * 2 + 3
	- 1 + (2 * 3) --> 1 + 2 * 3
	- (1 + 2) * 3 --> (1 + 2) * 3
	- 1 * (2 + 3) --> 1 * (2 + 3)
	- Summary: (1 * 2) + ((1 + 2) * (3 + 4))
	           --> 1 * 2 + (1 + 2) * (3 + 4)
- add < div
	- (1 / 2) + 3 --> 1 / 2 + 3
	- 1 + (2 / 3) --> 1 + 2 / 3
	- (1 + 2) / 3 --> (1 + 2) / 3
	- 1 / (2 + 3) --> 1 / (2 + 3)
	- Summary: (1 / 2) + ((1 + 2) / (3 + 4))
	           --> 1 / 2 + (1 + 2) / (3 + 4)
- sub < mul
	- (1 * 2) - 3 --> 1 * 2 - 3
	- 1 - (2 * 3) --> 1 - 2 * 3
	- (1 - 2) * 3 --> (1 - 2) * 3
	- 1 * (2 - 3) --> 1 * (2 - 3)
	- Summary: (1 * 2) - ((1 - 2) * (3 - 4))
	           --> 1 * 2 - (1 - 2) * (3 - 4)
- sub < div
	- (1 / 2) - 3 --> 1 / 2 - 3
	- 1 - (2 / 3) --> 1 - 2 / 3
	- (1 - 2) / 3 --> (1 - 2) / 3
	- 1 / (2 - 3) --> 1 / (2 - 3)
	- Summary: (1 / 2) - ((1 - 2) / (3 - 4))
	           --> 1 / 2 - (1 - 2) / (3 - 4)
- mul = div
	- (1 / 2) * 3 --> 1 / 2 * 3
	- 1 * (2 / 3) --> 1 * 2 / 3
	- (1 * 2) / 3 --> 1 * 2 / 3
	- 1 / (2 * 3) --> 1 / (2 * 3)
	- Summary: (1 / 2) * ((1 * 2) / (3 * 4))
	           --> 1 / 2 * 1 * 2 / (3 * 4)
